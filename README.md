# IPS-Ambilight-Control

IP-Symcon-Modul zur gemeinsamen Steuerung von WLED, HyperHDR und einer Apple-TV-Statusquelle auf dem Raspberry Pi.

## Version 0.6.1

- WLED: Status, Ein/Aus, Helligkeit und Presets
- HyperHDR: Serverstatus und Komponentensteuerung
- Apple TV: Abfrage des Raspberry-Monitors über HTTP
- Automatik: `playing` -> Live, `paused`/`idle` -> Pausenmodus, `standby`/`offline` -> Aus

## Installation

Repository in IP-Symcon aktualisieren und die vorhandene Instanz erneut öffnen. Die bestehenden Library- und Modul-GUIDs wurden beibehalten.
