# Apple-TV-Monitor

Der bereits eingerichtete Raspberry-Dienst stellt den Apple-TV-Status unter `http://<Raspberry-IP>:8091/status` bereit. Die Modulkonfiguration erwartet dieselben Felder wie der im Projektverlauf erstellte `app.py`-Dienst: `online`, `power`, `state`, `app`, `title`, `media_type`, `updated` und `error`.
